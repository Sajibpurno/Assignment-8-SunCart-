# 🚀 SunCart - Premium Digital Asset Marketplace

**SunCart** is a modern, high-performance web application designed for creators and professionals to discover and purchase premium digital assets. Built with a focus on speed, security, and a utility-first design approach.

---

## 🔗 Live Demo
https://assignment-8-sun-cart.vercel.app/

---

## 📖 Purpose
The primary goal of SunCart is to provide a seamless and secure platform for users to access high-quality digital tools. Whether you are a developer, designer, or digital entrepreneur, SunCart simplifies the process of finding and acquiring the assets you need to scale your projects.

---

## ✨ Key Features
*   **Secure Authentication:** Robust login and signup system powered by `better-auth`.
*   **Role-Based Access:** Protected routes (like `/my-profile`) using custom Next.js middleware for enhanced security.
*   **Dynamic Marketplace:** A fully functional digital product showcase with category filtering.
*   **Responsive UI:** A mobile-first, modern interface built with the latest Tailwind CSS 4.
*   **Blazing Fast Performance:** Leverages Next.js App Router and Vite for optimized loading speeds.
*   **Premium Design:** Clean and professional "About Us" and Landing pages tailored for a tech startup vibe.

---

## 🛠️ Technology Stack & NPM Packages

### Core Frameworks:
*   **Next.js / React.js (Vite):** Frontend architecture.
*   **Tailwind CSS 4:** Styling and responsive design.

### Key Dependencies:
*   **`better-auth` / `auth`:** For secure session management and authentication.
*   **`lucide-react`:** For premium, lightweight icons.
*   **`framer-motion`:** For smooth UI animations and transitions.
*   **`clsx` & `tailwind-merge`:** For dynamic class handling in Tailwind.
